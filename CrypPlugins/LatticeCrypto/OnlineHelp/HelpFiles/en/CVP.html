Closest Vector Problem (CVP)

The closest vector problem is beside the Shortest Vector Problem an interesting problem associated with lattices. A target point given, which is located within the vector space, but does not necessarily have to be a lattice point itself. CVP is looking for a lattice point, which is close to this point. This means the distance of both points has to be minimal.

This is interesting for example in connection with the GGH Cryptosystem, which places its safety in dissolving the CVP.

In this plugin the visualization is only possible in dimension 2.
