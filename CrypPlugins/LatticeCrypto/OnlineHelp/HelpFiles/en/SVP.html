Shortest Vector Problem (SVP)

The shortest vector problem is beside the Closest Vector Problem an  interesting problem associated with lattices. One searches for a vector that belongs to a lattice and has minimum length. That means there may not be another vector (except the zero vector), which is shorter.

To get this shortest vector the basis of the lattice must be reduced if necessary. This is done for example with the Gauss algorithm (in two-dimensional case) or the LLL algorithm.
