Successive Minima

The successive minima are the lengths of the reduced vectors. They are designated with λ and ordered. The first minimum λ1 thus refers to the shortest vector v1. Let the minima be defined as follows:

The definition can be understood as a sphere of radius r which "inflates" until it contains all n successive minima sorted by length.

The Hermite constant γ allows an upper bound for a vector v with minimum length. Exact values ​​for γ are only known for the dimensions 1, 2, ... , 8 and 24. For the second dimension, it is about 1.26. Using the Hermite constant one can construct a circle which contains the shortest vector (see graph below).
