Pari/GP for the evaluation of the logarithmic integral

Pari/GP is an open source computer algebra system (CAS), that offers very fast implementations of arithmetic functions.
It evaluates the logarithmic integral with much more precision than using Simpson's rule.
Pari/GP can be downloaded from the webite of the University of Bordeaux.
Activate the checkbox "Pari/GP" for using Pari for the evaluation of the logarithmic integral and select the program "gp.exe" in the file dialog. The program is in included the Pari/GP installation folder.
The CrypTool script describes in chapter "3.13 Appendix: Visualization of the quantity of primes in higher ranges" functions for estimating the number of primes in more detail, and how they can be calculated with the CAS Sage. In the meantime, Pari/GP is part of Sage.
